About Site

This Web Application is designed for visualisation purposes of Sorting Algorithms. Sorting Algorithms supported at the moment are: Bubble Sort, Selection Sort, Merge Sort, Quick Sort, Insertion Sort. Dynamic array size for visualisation, Speed control for the Sorting process, button to generate a new randomized array are some of the features of the Web Application.
